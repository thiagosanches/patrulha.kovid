# patrulha.kovid

### Introdução
Simples bot que navega no 'explore' do instagram e comenta em perfis abertos. A ideia é conscientizar a galera de uma forma gentil.

### Como executar?
```bash
node main.js false <usuário instagram> '<senha instagram>'
```
